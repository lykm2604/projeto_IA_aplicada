# Python e IA
Projeto desenvolvido durante a realização do curso "Python: Inteligência Artificial Aplicada" da Alura.

Programa em python que recebe um arquivo de texto contendo avaliações de uma aplicação e, então, envia separadamente cada uma para análise feita por uma inteligência artificial em um servidor local. O programa retorna o número de avaliações positivas, negativas e neutras junto de uma lista com o nome do cliente, avaliação no idioma original, tradução para o pt-br, e o sentimento relativo da avaliação.

## Processo

Foi utilizado o Visual Studio Code para a organização dos arquivos e programação. O LM Studio para rodar localmente o LLM utilizado.

## Recursos

O LLM utilizado foi o **Google Gemma** na sua versão **google/gemma-3-4b**
Também foi importada a biblioteca **openai** para vincular a IA ao projeto
